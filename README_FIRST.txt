PLAYCONNECT - CONNECTED GAMES PLATFORM

2. Leggere START_HERE.md.
3. Avviare Docker Desktop.
4. Windows: doppio clic su start-demo.bat.
5. macOS/Linux: eseguire ./start-demo.sh.
6. Applicazione: http://localhost:8080
